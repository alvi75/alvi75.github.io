Literature Review of PEFT in SE

Systematic literature review and implementation of PEFT techniques for large code models

A comprehensive systematic literature review examining parameter-efficient fine-tuning (PEFT) methods for large code models across software engineering tasks. This work analyzes 28 peer-reviewed papers, identifying patterns in configuration strategies and adaptation trade-offs across code generation, summarization, and translation tasks.
